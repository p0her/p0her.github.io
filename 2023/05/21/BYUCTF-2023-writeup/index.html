shellcode

Give r, w, x to the 0x777777000 address, and can input string to 0x777777000. and execute it. so this is shellcode challenge!
First, there is a 20 byte difference(0x777777000, 0x777777020, …), so you have to put in a shell code that jumps next 20 bytes.
Initially, I tried to run execve(“/bin/sh”, 0, 0) at once, but it was difficult because the length I could type was too short.
So I got one more input to 0x777777000 and entered execve(“/bin/sh”, 0, 0) shell code.
